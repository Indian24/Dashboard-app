# TabBlazor

TabBlazor is a lightweight and modern admin dashboard template for Blazor applications, inspired by the Tabler UI framework.
It is designed to help developers quickly build clean, responsive, and maintainable dashboards using ASP.NET Core Blazor while keeping JavaScript usage to a minimum.

---
## ✨ Features

-Modern Tabler-based UI design
-Built with ASP.NET Core Blazor
-Minimal JavaScript usage
-Fully responsive dashboard layout
-Reusable and modular UI components
-Clean and maintainable project structure
-Easy to customize and extend for new features

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/Indian24/Dashboard-app.git
```

Navigate to the project

```bash
cd Dashboard-app
```

Run the application

```bash
dotnet run
```

Open in browser

```
https://localhost:5001
```
